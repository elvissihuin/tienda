Bienvenido a **Tu Estilo**, tu tienda de ropa online con las últimas tendencias. Este proyecto es una plataforma de comercio electrónico desarrollada para facilitar la compra y venta de indumentaria de forma rápida, segura y atractiva.

---

## ✨ Características principales

- 🛍️ **Catálogo dinámico** – Navegación por categorías (remeras, pantalones, vestidos, accesorios).
- 🔍 **Búsqueda avanzada** – Filtros por talla, color, precio y temporada.
- 🛒 **Carrito de compras** – Añade, elimina y modifica cantidades antes de pagar.
- 💳 **Pasarela de pago** – Integración con MercadoPago / PayPal (simulada en desarrollo).
- 👤 **Perfil de usuario** – Historial de pedidos, lista de deseos y datos personales.
- 📱 **Diseño responsive** – Experiencia perfecta en móviles, tablets y desktop.
- 🌐 **Multi-idioma** – Soporte para español e inglés.

## 🛠️ Tecnologías utilizadas

**Frontend:**
- React + Vite ⚡
- Tailwind CSS para estilos modernos 🎨
- Redux Toolkit (manejo de estado) 🔄

**Backend:**
- Node.js + Express 🚀
- MongoDB con Mongoose (base de datos) 🍃
- JWT para autenticación 🔐

**Otros:**
- Cloudinary para imágenes 📷
- Stripe / MercadoPago (sandbox) 💰
- Docker para entorno de desarrollo 🐳
